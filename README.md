# Coding Practice Repository

This repository contains practice programs across multiple languages, focused on core programming concepts like arrays, loops, OOP, patterns, and basic problem solving.

## Languages Covered

- C++
- C
- Java
- Python
- JavaScript
- HTML

## Project Structure

```text
coding/
|-- c++/                # C++ practice programs
|-- c programing/       # C language practice programs
|-- java/               # Java source and class files
|-- python/             # Python scripts
|-- JavaScript/         # JavaScript files
|-- HTML/               # HTML files
\-- *.cpp, *.exe        # Additional C++ files in root
```

## Topics Practiced

- Arrays and 2D Arrays
- Vectors
- Loops and conditionals
- Functions
- OOP basics (class, constructor, static, operator)
- Matrix operations
- Pattern printing
- Number system and basic math logic

## How to Run

### C++

```bash
g++ filename.cpp -o output
./output
```

### C

```bash
gcc filename.c -o output
./output
```

### Java

```bash
javac FileName.java
java FileName
```

### Python

```bash
python filename.py
```

### JavaScript

```bash
node filename.js
```

## Notes

- This is a learning repository, so file names and topics are practice-oriented.
- Some compiled output files (`.exe`, `.class`) are also present.
